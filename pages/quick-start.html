# Quick Start

Creating a Nuek Component

In this section, I would be putting you through on how to create and use a basic Nuek Component in your app.

Using npm/pnpm

You should now have Nuekjs installed on your project. To create a new Nuek component, let's create a JS file and instance a NuekComponentObject().

Now you need to create the NuekJS component to that filePath...

  • Templating System: Nuekjs allows markup representation through a simple and intuitive templating system for each component in Single File Components(SFCs).
  • Resuability: Nuekjs allows its components to be reused across the codebase, so you can store this component data in a variable and reference later across your codebase.
  • Reactivity and Props: Nuekjs's PropComponents allow for a dynamic flow of data in the components either static or dynamic and DynamicComponents compliment this and every other feature by making component data States reactive.
  • Server Side Rendering(SSR): Nuekjs supports optional SSR features to it's components to improve state of data across your application.
